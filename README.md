# Application de Gestion de Stock - Infirmerie SCG

## 📋 Description
Application web locale pour gérer le stock de médicaments de l'infirmerie SCG.

## 🚀 Installation

1. Créer un dossier `infirmerie-scg`
2. Copier les 3 fichiers : `index.html`, `styles.css`, `app.js`
3. Ouvrir `index.html` dans un navigateur web

## ✨ Fonctionnalités

- ✅ Ajout/Modification/Suppression de médicaments
- ✅ Ajustement rapide des quantités
- ✅ Alertes stock faible
- ✅ Alertes dates d'expiration
- ✅ Recherche et filtres avancés
- ✅ Export CSV
- ✅ Stockage local (données persistantes)
- ✅ Interface responsive
- ✅ Statistiques en temps réel

## 💾 Stockage
Les données sont sauvegardées localement dans le navigateur (localStorage).

## 📊 Export
Possibilité d'exporter la liste complète en format CSV.
